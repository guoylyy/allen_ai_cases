# Day 1: 外贸网站问询智能体

## 🎯 项目目标

构建一个外贸网站问询智能体，帮助外贸站点的客户通过图片来查询是否有对应商品售卖。该智能体能够：
1. 接收用户上传的商品图片
2. 分析图片内容，识别商品特征
3. 在商品数据库中搜索匹配的商品
4. 返回查询结果（是否有货、价格、规格等信息）

## 📋 功能需求

### 核心功能
1. **图片上传接口**：支持用户上传商品图片
2. **图片分析模块**：使用AI模型识别图片中的商品特征
3. **商品搜索模块**：根据识别特征在数据库中搜索匹配商品
4. **结果返回模块**：以友好格式返回查询结果

### 扩展功能
1. **多语言支持**：支持中英文查询
2. **相似商品推荐**：当完全匹配不存在时，推荐相似商品
3. **价格比较**：提供市场价格比较
4. **库存状态实时更新**：连接库存管理系统

## 🏗️ 系统架构

### 技术栈选择
- **前端**：React + TypeScript（用户界面）
- **后端**：Python FastAPI（API服务）
- **AI模型**：OpenAI CLIP / Google Vision API（图片识别）
- **数据库**：PostgreSQL（商品信息存储）
- **部署**：Docker + AWS/GCP

### 架构设计
```
用户界面 (React) → API网关 (FastAPI) → 图片处理服务 → AI识别服务 → 商品数据库
                                     ↓
                               结果返回服务
```

## 🔍 查询流程

1. **用户上传图片**
2. **系统提取图片特征**
   - 使用AI模型识别商品类别
   - 提取颜色、形状、纹理等特征
   - 识别品牌和型号（如果可见）
3. **特征匹配**
   - 将提取特征与数据库商品特征对比
   - 计算相似度分数
   - 返回Top N匹配结果
4. **结果展示**
   - 显示匹配商品信息
   - 提供购买链接或联系方式
   - 推荐相似商品
